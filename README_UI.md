# ARISE_osc — UI (Dark theme)

This folder contains a minimal Vite + React UI and a simple Node/Express server to serve the built files.

Quick start

- Install dependencies:

```
npm install
```

- Run development server (Vite):

```
npm run dev
```

- Build and serve production (Express):

```
npm run build
npm start
```

Notes

- The navbar is intentionally transparent (no separate background) to keep the layout plane clean.
- Responsive behavior: desktop shows inline nav; mobile shows a hamburger which toggles the nav.
